# Paperclip Setup

Guides for setting up [Paperclip](https://github.com/paperclipai/paperclip) — an AI agent platform.

## Setup guides

- **[Docker](DOCKER.md)** — Run Paperclip in a Docker container (macOS). Recommended for developers or anyone comfortable with the terminal.
- **[Windows](WINDOWS.md)** — Install Paperclip directly on a Windows computer. Written for Swedish-speaking non-developers, with references to the Swedish Windows UI.
